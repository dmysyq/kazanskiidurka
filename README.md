### Краткое описание вариантов

- **Вариант 1**: читает одну строку текста, убирает пунктуацию, считает частоту слов и выводит их в алфавитном порядке.
- **Вариант 2**: читает несколько строк до пустой строки, считает общее число слов и число различных слов.
- **Вариант 3**: для одной строки текста считает общее и уникальные слова и вычисляет коэффициент лексического разнообразия.
- **Вариант 4**: строит мини-словарь соответствий по введённым парам и по запросу выдаёт перевод или сообщение «Нет в словаре».
